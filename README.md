# web-zcjc-guide

本仓库用于归档和发布多个独立 HTML 页面，内容涉及特定主题的资料整理与展示。仓库本身不针对任何单个域名或具体网站，仅为静态页面集合提供存放与访问入口。

## 项目简介

- 目标：提供一个统一存放 HTML 页面的仓库，便于浏览、维护与分享。
- 范围：页面内容覆盖多个独立主题，彼此之间无强关联。
- 性质：非商业项目，仅作为资料归档与静态页面托管使用。

## 目录说明

```
web-zcjc-guide/
├── index.html          # 首页 / 页面索引
├── pages/              # 存放所有 HTML 页面文件
│   ├── example-page-1.html
│   ├── example-page-2.html
│   └── ...
└── assets/             # 静态资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：可选的入口页面，用于导航或说明。
- `pages/`：核心内容目录，每个 HTML 文件为一个独立页面。
- `assets/`：页面引用的样式、脚本、图片等资源文件。建议按类型分目录存放。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开。
- 页面之间通过相对路径引用资源，迁移时需保持目录结构一致。
- 页面内容以客观资料整理为主，不包含商业推广或诱导点击元素。
- 归档页面可能不定期更新、补充或移除。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务发布后访问（需自行配置）。
